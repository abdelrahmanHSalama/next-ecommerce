## NextShop ([Live Preview](https://abdelrahman-next-ecommerce.vercel.app/))

ECommerce Website built using Next.js and TypeScript.

**✨ Features:**

🔍 Product listing.

🧾 Dynamic product detail pages.

🛒 Shopping Cart.

❤️ Wishlist.

🛍 Orders & Checkout.

**⚡ Tech Stack:**

- Next.js

- TypeScript

- NextAuth

- TanStack Query

- Zustand

- Tailwind
